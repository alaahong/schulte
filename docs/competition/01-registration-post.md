## 【正文】

### 1. 创意名称 + 创意介绍

- **想解决什么问题**
  - 舒尔特方格(Schulte Grid)是 60 多年验证有效的注意力训练法,但**市面产品普遍存在三大痛点**:
    1) 广告多 / 强制注册 / 收费门槛高,真正想练的人被劝退;
    2) 只在某一端可用,iPad / 电脑 / 手机之间体验割裂、行为不一致;
    3) 玩法单一,长期训练容易产生厌倦感,坚持不下来。
  - 因此想做一个**零广告、零注册、零付费、跨端可用、可高度定制**的舒尔特方格工具,让"每天 5–10 分钟练专注力"这件事真正可行。

- **为什么会想到做这个**
  - 家里有正在读书的小朋友,每天放学要做舒尔特训练;我自己也是知识工作者,工作间隙也需要短时训练;
  - 自己搜索了一圈发现:**iPad / 电脑端能用的几乎都是付费或要注册,免费版充斥着弹窗广告;玩法上也很单一,孩子做几次就失去兴趣**;
  - 既然 TRAE 可以帮我把脑中的产品想法真正落地,那就动手把它做出来,顺便参赛。

- **大概是什么产品**
  - 基于 **Tauri 2 + Vue 3 + Vite + TypeScript + Pinia + Tailwind + Dexie** 构建的**跨平台舒尔特方格专注力训练应用**,
  - 一份代码可打包为 **Web / PWA / Windows / macOS / Linux 桌面,以及 iOS / Android 原生应用**,共 7 端。

### 2. 目标用户及痛点

- **核心用户群体**
  - 7–18 岁学生及其家长(注意力训练刚需);
  - 18+ 知识工作者 / 程序员 / 备考党(碎片化时间训练);
  - 培训老师 / 学校(教学场景嵌入);
  - 老年认知训练用户(需大字号、低干扰、可调难度)。

- **典型使用场景**
  - 放学后 / 写作业前 5–10 分钟;
  - 工作学习间隙 1–3 分钟短时训练;
  - 课堂上 / 培训机构的白板/电脑/平板;
  - 通勤地铁上手机快速训练一局。

- **当前痛点(若没有本产品)**
  - 广告弹窗打断训练节奏;
  - 强制注册、付费后才能解锁完整功能;
  - 跨端不通用,iPad 进度与手机不通;
  - 玩法单一,孩子做几次就厌倦;
  - 无按年龄段的成绩评估,只能凭感觉判断进步。

### 3. 价值与意义

- **教育价值**: 把经过 60 多年心理学验证的舒尔特方格方法**数字化、可量化、可渐进**,并按年龄段自动评级(优秀 / 良好 / 中等 / 较弱);
- **效率价值**: 一份代码 7 端,避免重复开发;**PWA 离线可用 + IndexedDB 本地存储**,打开即用、零注册零广告;
- **社会价值**: 提供**完全免费、无干扰**的工具,帮助有需要的家庭与个人坚持训练;核心游戏逻辑可开源,鼓励教育领域二次开发;
- **创新价值**: 在传统舒尔特基础上,加入 **5 种训练模式(经典升序 / 反向降序 / 交替 / 红黑双驱 / 色彩模式)**,以及
  - **多色调色板(6 色,至少 2 色可多选)**,
  - **多彩色块(纯色 / 多彩,数字模式可切换)**,
  - **不规则形状(方块 / 圆润 / 正圆 / 异形)**,显著提高可玩性与可定制性,降低重复训练的厌倦感。
